PROYECTO FINAL (TIENDA)
Este es una aplicacion web en Django en formato de Tienda.

En esta se trabajo:

Herencia de Templates
Modelo principal tipo “producto/pagina”
Sistema de usuarios y perfiles
Manejo de saldo y compras ficticias
Reseñas con like/dislike
Sistema de mensajeria entre usuarios amigos+chat
Seccion de Blog con paginas informativas (FAQ, Soporte, Guia de compra)
Tecnologias utilizadas

python 3.13.x
Django 5.2.x
SQLite3
Django-ckeditor
HTML, CSS
Pasos para instalar
Clonar el repositorio
Crear venv
instalar dependencias: pip install -r requirements.txt
Aplicar migraciones: pythonmanage.py migrate
Crear superusuario
Ejecutar el servidor: python manage.py runserver
